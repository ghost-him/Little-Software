## 密码生成器

用于生成随机数的密码

可以设置的参数有：

* 密码的长度
* 密码的种类
	* 纯数字
	* 纯小写字母
	* 数字,小写字母混合
	* 纯大写字母
	* 大写字母,数字混合
	* 小写字母,大写字母混合
	* 数字,小写字母,大写字母混合
* 生成密码的数量
* 生成的路径（控制台、文件）
